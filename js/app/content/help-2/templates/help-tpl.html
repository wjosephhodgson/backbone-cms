Change Homepage Layout

When you first signed up for eFlorist, you selected a homepage layout from a list of available options. Luckily, you're not limited to that template. You can adjust both the layout and number of items featured on your homepage at any time.
Go to Merchandising, then Homepage Settings.
On the right-hand side, you'll see previews for all homepage layouts available to you.
Click to see them slightly enlarged.
Select the homepage layout you want to use.
Review the Featured Item Count section underneath the Current Homepage Layout preview. Each layout has a different number of components and featured products.
When you select one, the Featured Item Count updates to match accordingly. We recommend keeping the suggested number, but you can adjust as needed, provided you don't exceed the maximum number for that layout.
Click the Save Changes button at the top right of the page. You can also view your new homepage in real-time - just click the See It Live link at the top of the page.
